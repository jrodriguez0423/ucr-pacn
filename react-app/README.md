# UCR PACN — React (Vite)

This is a minimal React + Vite starter for the UCR PACN site.

Setup (Windows):

```bash
cd react-app
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).

Notes:
- Pages: Home, About, Tickets, Merch, Contact.
- Ticketing and merch are placeholders; integrate a payment provider later.
- Replace `public/images/*` with real photos.
